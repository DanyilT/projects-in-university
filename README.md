## ‚úÖ Assignment Brief Summary
For this module, the requirements were:

### **Required Pages**
- Home page (flexbox layout)
- Contact page
- Site map (nested list with links)
- Characters / staff / members page (plus subpages)
- Merchandise page (A‚ÄìZ list, with clickable images)
- Additional pages to reach at least **9 total**

### **Home Page Requirements**
- Flexbox layout
- Horizontal & vertical navigation bar
- Text about the chosen topic
- External web link
- Valid image
- Footer with Facebook and X(Twitter) icons linking externally

### **Remaining Pages Requirements**
- Contact page with email and phone link
- Characters/staff page + at least 2 detailed subpages
- Merchandise page with at least 4 items, 2 having detailed pages
- All text must be original

### **Wireframe Requirement**
- 3 page wireframes using Mockflow / Figma / Draw.io
- Home page + 2 additional pages
- Exported to Word document (`wireframe.doc`)

### **General Coding Requirements**
- External CSS file(s)
- Proper indentation and comments
- Functional internal, external, image, and email links
- Valid HTML5 & CSS3 (validator required)
- Responsive layout across devices
- Appropriate titles, alt tags, and consistent navigation
- Well-designed, user-friendly appearance
